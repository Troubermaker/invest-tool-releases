# K 线技术分析指标使用指南

> 本工具内置 16 项 K 线技术指标 + 2 个智能扫描器，分**基础**和**高级**两个层级。
> 基础层日常使用足够；高级层是为深度研究/短线介入设计的，需管理员权限解锁。

---

## 0. 用户角色与指标分级

| 层级 | 可见内容 |
|---|---|
| **普通用户** | 基础指标（均线 / BOLL / MACD / KDJ）+ K 线 drawer + 自选 + 持仓 + 行情页 |
| **管理员** | 以上全部 + 全部高级指标（共振 / 三维启动 / 主升 等）+ 找候选 / 找发车扫描器 + pytdx 数据源 |

**解锁管理员**：在任意页面按 `Ctrl + Shift + A` → 输入管理员密码 → 解锁。
再次按 `Ctrl + Shift + A` 可退出管理员模式。

## 1. 速览（按打开 chip 在 K 线 drawer 中的顺序）

### 基础（所有用户）

| chip | 视觉形式 | 表达内容 |
|---|---|---|
| **均线 MA** | 5/10/20 三条彩色折线 | 短/中/长期趋势 |
| **BOLL 布林带** | 三条线 + 中间填充 | 中轨 = 20 日均价；上下轨 = ±2σ |
| **MACD** | 副图：DIF/DEA 双线 + 红绿柱 | 趋势动能 + 金叉死叉 |
| **KDJ** | 副图：K/D/J 三线 | 超买超卖 + 短线拐点 |

### 高级（仅管理员）

| chip | 视觉形式 | 表达内容 |
|---|---|---|
| 压/支 SR | 横向直/虚线（红 = 压、绿 = 支） | 近 N 根的最高/最低 + 聚类强档位 |
| 箱体 BOX | 蓝紫半透明矩形 | 持续震荡 / 强势平台 |
| 通道 CHAN | 青色平行四边形 | 价格运行轨道 + 上下趋势线 |
| 波段 ZZ | 折线骨架 + 节点 | 主结构 / 当前在第几波 |
| 斐波那契 FIB | 7 条玫红横线 | 主波段 0-100% 回撤位 |
| 形态 PAT | 箭头 + 文字 | 锤/星/吞/十 5 种敏感形态 |
| **🚀 主升启动 RALLY** | 蓝色矩形 + 突破竖线 + 交易计划 banner | 长横盘后突破 |
| **🎯 三维启动 TRIPLE** | 蓄势框 + 试盘竖线 + 突破竖线 + 完整交易计划 | **蓄势→试盘→突破** 完整序列识别 |
| **缺口 GAP** | 半透明横向条带 | 跳空缺口（向上/向下）|
| **共振 RESO** | 右上角红边框面板 | **多源信号汇聚的高质量价位** ⭐ |
| **📊 启动分 SCORE** | 副图：评分曲线 | 主升浪概率综合评分 (0-100) |
| **📢 信号 SIG** | 散点 + 标注 | 综合信号汇总 |

色弱友好原则：所有指标统一通过**颜色 + 线型 + 形状**三重冗余编码。
- 蓝 / 红 = 上行 / 升势 / 多头 / 支撑
- 橙 / 绿 = 下行 / 跌势 / 空头 / 压力（注：A 股惯例红涨绿跌，部分横向辅助线沿用此约定）
- 实心圆 = 高点，空心圆 = 低点
- ↑ 三角 = 看涨，↓ 三角 = 看跌

## 2. 基础指标详解

### 2.1 均线 MA

**默认显示** MA5（短线）/ MA10（次短）/ MA20（中线）。

| 配色 | 周期 | 含义 |
|---|---|---|
| 暗金 | MA5 | 短期最敏感线 |
| 墨青 | MA10 | 次短线 |
| 沉紫 | MA20 | 中期生命线 |

**实战**：
- **多头排列**（MA5 > MA10 > MA20）→ 上升趋势确立
- **空头排列**（MA5 < MA10 < MA20）→ 下跌趋势确立
- MA 粘合 + 缠绕 → 横盘 / 蓄势中
- 价格回踩 MA20 不破 → 强支撑
- MA20 由下向上穿 MA60 → 中长线趋势反转（俗称"金叉"）

### 2.2 BOLL 布林带

中轨 = 20 日均价；上轨 = 中轨 + 2σ；下轨 = 中轨 - 2σ。
σ = 20 日收盘价标准差。

**实战**：
- 价格触上轨 → 短期超买，可能回落
- 价格触下轨 → 短期超卖，可能反弹
- 上下轨**收口**（带宽收窄）→ 行情即将选择方向（**Bollinger Squeeze**）
- 上下轨**张口**（带宽扩张）→ 趋势启动中
- 价格沿上轨持续运行 → 强势单边市，**别轻易抢顶**

### 2.3 MACD

**指标**：
- DIF = EMA12 - EMA26（快线）
- DEA = DIF 的 9 日 EMA（慢线）
- 柱状（HIST）= DIF - DEA，即"两条线的差距"

**实战**：
- DIF 上穿 DEA → **金叉**（买入信号）
- DIF 下穿 DEA → **死叉**（卖出信号）
- 红柱放大 → 多头力量增强
- 绿柱缩短 → 空头力量减弱
- 价格创新高但 DIF 没创新高 → **顶背离**（重要警示）

### 2.4 KDJ

**指标**：基于近 9 根 K 线最高最低价的 stochastic oscillator。

**实战**：
- J > 100 → 超买（短期回调风险）
- J < 0 → 超卖（短期反弹机会）
- K 上穿 D → 短线买入
- K 下穿 D → 短线卖出
- KDJ 比 MACD 灵敏，适合做超短线

## 3. 形态识别（管理员）

### 3.1 压力/支撑（两种模式同时打开）

**简单版**：近 60 根（日K）的最高 = 压力，最低 = 支撑。
**聚类版**：滑窗找所有 swing 高/低 → ±1.5% 内归并 → 至少触及 2 次 → 上下各取 Top 3。

**视觉对比**：

| | 简单版 | 聚类版 |
|---|---|---|
| 线宽 | 1px（细） | 2px（粗，醒目） |
| 线型 | 红虚线 / 绿点线 | 红实线 / 绿实线 |
| 标题 | `压·近60` | `强压·×7`（带触及次数） |
| 含义 | 近期边界 | **历史多次被验证的真档位** |

**两者重合的位置 = 超强档位**（共振面板会自动识别）。

### 3.2 箱体 BOX

滑窗扫描，连续 ≥8 根 K 线满足 `(高-低)/中位 ≤ 12%` → 视为震荡区。

**实战**：
- 箱内：高抛低吸
- 突破箱顶 + 放量：目标 ≈ 箱顶 + 箱高
- 跌破箱底 + 放量：目标 ≈ 箱底 - 箱高
- 越窄越久 → 突破后能量越强（这就是 3.3 平台的进阶版）

### 3.3 平台（箱体的强化版，已合并到箱体识别中）

满足"≥20 根 + 振幅 ≤10% + 缩量 ≥15%"的箱体会被识别为平台，颜色更深（青绿）。
代表"价格横盘 + 资金观望或暗中吸筹"，**带量突破才可靠**。

### 3.4 通道 CHAN

取最近 zigzag 的 swing 高低 → LR 拟合斜率均值 → 上下严格平行通道。

**实战**：
- 贴上轨 = 高位（潜在卖点）
- 贴下轨 = 低位（潜在买点）
- 突破上轨 + 量能 = 趋势加速
- 通道斜率 = 主趋势方向与速度

### 3.5 波段 ZZ（Zigzag）

大窗口（±10 根）找 swing → 高低交替化 → "高→低→高→低"折线骨架。

**实战**：
- 看清主结构 / 当前在第几波
- 历史每段涨跌幅可作为下一段参考
- 连续创新高但 MACD 走低 = 顶背离

### 3.6 斐波那契回撤 FIB

复用 zigzag 节点找最大振幅段（≥10%），画 7 条经典回撤线：**0 / 23.6 / 38.2 / 50 / 61.8 / 78.6 / 100 %**。

| 回撤档位 | 含义 |
|---|---|
| 23.6% | 浅回撤，强势行情 |
| **38.2%** | 多头健康调整（黄金买点之一） |
| 50% | 中性 |
| **61.8%** | 深度回调（黄金分割位，最经典买点） |
| 78.6% | 极深，多头转空风险 |

**最强用法 = 与其他工具共振**：
- Fib 38.2% **+** MA20 → 强支撑
- Fib 61.8% **+** 通道下轨 → 超强支撑
- Fib 50% **+** 上升趋势线 → 教科书买点

### 3.7 K 线形态 PAT

5 种敏感形态：

| 形态 | 判定 | 信号 |
|---|---|---|
| **锤子线** | 下影 > 实体×2.2，上影 < 实体×0.4 | 底部反转买入 |
| **射击之星** | 上影 > 实体×2.2，下影 < 实体×0.4 | 顶部反转卖出 |
| **看涨吞没** | 前阴后阳，后实体完全包裹前实体 | 反转向上 |
| **看跌吞没** | 前阳后阴，后实体完全包裹前实体 | 反转向下 |
| **十字星** | 实体 < 振幅×8% | 犹豫/反转预警 |

**关键**：形态信号**单独看不够**，必须结合**位置**：

| 形态 + 位置 | 含义 |
|---|---|
| 锤子线 + 在支撑 / Fib 位 / 通道下轨 | **强买入信号** |
| 射击之星 + 在压力 / Fib 位 / 通道上轨 | **强卖出信号** |
| 看涨吞没 + 多次试探支撑后 | **底部反转确认** |

形态出现在**共振区**时 → 信号可靠度倍增。

### 3.8 ⚡ 共振检测 RESO（短线最关键）

把所有活跃指标产生的"当前关键价位"汇集起来，做价格聚类，找出**多源汇聚的"共振区"**。

**算法**：
1. 收集所有活跃指标的当前价位（MA/BOLL/SR/通道/箱体/Fib/趋势线 等十几种）
2. 价差 ≤ 1.2% 内归并到同一档
3. 不同 source 数 ≥ 2 才算共振
4. 按 source 数倒序，取 Top 6

**视觉**：主图右上角红框面板。
- 每条：价格 + 距现价 ±X% + ×N（来源数）+ source 列表
- 红色价格 = 高于现价（潜在压力）
- 绿色价格 = 低于现价（潜在支撑）

**为什么共振是杀手级功能**：
单一指标信号容易出现假信号，多源指标汇聚到同一价位 = 不同分析框架对同一结论的独立验证。**短线最怕假信号，共振是最有效的过滤器**。

| 共振组合 | 解读 |
|---|---|
| 简单支撑 + Fib 50% + MA20 | 三重支撑，回踩到此 = 高胜率买点 |
| 强压 + Fib 61.8% + 通道上轨 | 三重压力，反弹到此 = 高胜率卖点 |
| 通道下轨 + 上升趋势线 + 平台底 | 三重支撑，主趋势未破 |

操作要点：
- ×3 以上的共振区 = 强信号
- 距现价 ±5% 内的共振区最有交易价值

## 4. 启动信号（管理员核心功能）

### 4.1 缺口 GAP

跳空缺口检测：
- **向上缺口**：今日最低 > 昨日最高（看涨）
- **向下缺口**：今日最高 < 昨日最低（看跌）

**视觉**：横向半透明色带，标注是否回补。
**实战**：未回补的向上缺口 = 重要支撑；未回补的向下缺口 = 重要压力。

### 4.2 🚀 主升启动 RALLY

**核心思想**："长期横盘 → 突破"这种最稀缺的形态。

**条件**：
1. 找到一段长横盘（≥30 根 + 振幅 ≤15%）
2. 横盘后向上突破上沿（或向下跌破下沿）
3. 突破后峰值幅度 ≥ 8%
4. 当前价仍在新趋势中（≥ upper × 0.97 / ≤ lower × 1.03）

**视觉**：横盘段画蓝色矩形 + 突破竖线 + 顶部交易计划 banner。

**交易计划基于横盘高度 H = upper - lower**：
- 入场 = 突破 K 收盘
- 止损 = 横盘反向沿 × 0.97
- 目标 1 = 入场 + H × **1.618**（斐波那契扩展）
- 目标 2 = 入场 + H × **2.618**

风险收益比一目了然。

### 4.3 🎯 三维启动 TRIPLE（最强大也最严格）

**核心思想**：必须按 **蓄势 → 试盘 → 突破** 时间顺序触发，每阶段满足 K 线 + MA + 量能多重条件。比主升启动严格得多，**召回低但精准**。

#### Stage 1 蓄势
| 条件 | 阈值 |
|---|---|
| 根数 | 20 ~ 120 根 |
| 收盘价中位数 ±振幅 占比 | ≥ 80% |
| 振幅自适应 | 高股性 ±20%，中 ±15%，低 ±10% |
| 振幅收窄 | 后半 / 前半 ≤ 0.85 |
| MA5/10/20 间距 / 中位数 | ≤ 7%（粘合）|
| **趋势位置过滤** | 蓄势均价 ≥ 近 250 根最低价 ×1.30（防阴跌假蓄势）|

#### Stage 2 试盘（蓄势结束后 25 根内）
两种型态任选其一：
- **长上影**：上影 ≥ 实体 × 1.5 且 量比 ≥ 2.5（射击之星型试盘）
- **长下影**：下影 ≥ 实体 × 1.5 且 量比 ≥ 1.3 且 收阳（金针探底型试盘）

附加确认：试盘后 25 根内**收盘不破 s2Low × 0.99**（破了视为失败）。

#### Stage 3 突破（试盘后 30 根内）
- 收盘突破 max(s1Upper, s2High)
- 量比 ∈ [1.3, 3.5]（**> 3.5 视为爆量诱多**）
- 实体 / 振幅 ≥ 0.55（实体饱满阳线）
- close > MA20 且 MA5 > MA20

#### 输出价位
- **黄金买点** = `min(试盘 K 中点, s1Upper)` — 突破后最稳健的回踩位
- **突破价** = s3 收盘价 — 已在车上的位置参考
- **止损价** = `s1Lower × 0.97` — 跌破蓄势下沿即形态破坏
- **isFresh** = 距今 ≤ 30 根

#### 视觉
蓄势框（半透明）+ 试盘竖线（"试 N前"）+ 突破竖线 + 左上角"三维启动" banner，含完整交易计划（黄金买点 / 追涨买点 / 加仓点 / 减仓线 / 止损 / 时间止损）。

### 4.4 📊 启动分 SCORE

副图评分曲线，对每根 K 线计算"主升浪概率" 0-100 分（5 维度加权）：

| 维度 | 满分 | 关键打分项 |
|---|---|---|
| 形态 | 25 | 突破创高 / 大阳线 / 实体饱满 |
| 量能 | 25 | 巨量 / 量价齐升 / 缩量涨警告 |
| 均线 | 25 | 多头排列 / 金叉 |
| MACD | 25 | 零轴上 / 金叉 / 红柱放大 |
| 共振 | 15 | 当日附近的多源共振区 |

实战：连续多根 ≥80 分 → 主升浪期；评分快速回落 → 趋势可能终结。

### 4.5 📢 信号 SIG

综合标记主升启动、三维启动、缺口、形态等所有触发点的散点视图，便于快速浏览历史关键节点。

## 5. 智能扫描器（管理员）

把"看 K 线"的逻辑放到批量扫描场景。流程闭环：

```
找候选 → 加自选 → 等触发 → 找发车 → 介入
```

### 5.1 大盘环境评分

所有扫描器都会在 modal 头部显示"大盘 强势/良好/震荡/弱势/破位 + 0-100 分"chip。
基于沪深 300 日 K 计算：
- close vs MA20 (±15 分)
- close vs MA60 (±20 分)
- MA20 / MA60 斜率 (±10/5 分)

**意义**：大盘破位时所有信号置信度自动打折（直接影响"找发车"评级里的"大盘分"维度）。

### 5.2 找候选 (Scan Candidates) — 板块联动股工具

**入口**：行情页 → 板块联动股表头 → 「找候选」按钮。

**用途**：在板块成分股里挑**还没启动**但形态符合的票，加入自选监控。

**算法**：
1. 拉取当前板块所有股票（剔除北交所）
2. 每只股票走 pytdx 拉日 K
3. 跑 `detectThreeStageLaunch`，过滤出 **currentStage ∈ {1, 2}**（未突破）
4. 排序：Stage 2 优先（试盘已成）→ 同阶段按 |距突破点距离| 升序

**输出列**：阶段 / 现价 / 突破线（s1Upper） / 距突破 / 蓄势天数 / 止损 / 「+ 加自选」按钮

**距突破标记（色弱友好双编码）**：
- ` · ` 中下沿（≤ -5%）
- `▲` 接近突破（-5% ~ -1.5%）
- `◆` **临门一脚**（-1.5% ~ 0）
- `⚠` 已超突破线（注意假突破）

### 5.3 找发车 (Scan Signals) — 多入口工具

**入口**：
- 自选页 → 「📊 扫描信号」
- 行情页热榜工具栏 → 「找发车」
- 行情页涨跌对比池工具栏 → 「找发车」（跌停池禁用）

**用途**：在已发车的票里挑**刚突破 0~30 天**的可追入信号。

**算法**：
1. 同找候选，但过滤条件改为 `isFresh && currentStage === 3`
2. 计算"距黄金买点距离" = (现价 - 黄金买点) / 黄金买点 × 100%
3. 调用 `gradeFreshSignal` 计算 S/A/B/C 评级

#### S/A/B/C 强度评级（满分 100）

| 维度 | 满分 | 关键打分项 |
|---|---|---|
| **形态** | 30 | 距黄金买点 ±1.5%（22 分）+ Stage 3 实体饱满（+8）|
| **量能** | 25 | 突破量比 1.5-2.5（15 分）+ 预突破温和放量铺垫（+10）|
| **均线** | 25 | close>MA20 && MA5>MA20（基础 10）+ MA5>10>20（+8）+ MA5>10>20>60 强势多头（+10）|
| **大盘** | 20 | 大盘评分 / 100 × 20 |

**评级阈值**：
- ≥ 85 → **S 级**（多重共振，可重仓）
- 70-84 → **A 级**（主力意图明显）
- 55-69 → **B 级**（信号成立有瑕疵）
- < 55 → **C 级**（观望）

**距买点标记**：
- `◆` 黄金买点附近（±1.5%）
- `▼` 折扣区（< -1.5%，现价低于买点）
- `▲` 追涨区（0 ~ 5%）
- `▲` 已涨过（> 5%，不建议追）

**实战工作流**：
1. 早盘看**热榜 + 涨停池 → 找发车**：哪些刚突破能追
2. 收盘后**板块联动 → 找候选**：把蓄势/试盘中的票加自选
3. 之后每天**自选 → 找发车**：等候选发车

## 6. 组合用法（实战）

### 6.1 三种典型行情匹配指标

| 行情 | 主力工具 | 辅助 |
|---|---|---|
| **趋势行情** | 趋势线 + 通道 + Fib | 波段、共振 |
| **横盘整理** | 箱体 + 简单 S/R | 强压/支、共振 |
| **转折期** | 波段 + 形态 + 共振 | 趋势线、Fib |

### 6.2 短线"看长做短"四步法 ⭐⭐⭐

#### Step 1：看长 — 确认主方向

打开：**通道 + 趋势线 + 波段**

- 通道斜率向上 + 蓝色趋势线还在 = **只做多不做空**
- 通道斜率向下 + 橙色趋势线还在 = **只做空或观望**
- 横盘 / 趋势模糊 → 跳过这只票

#### Step 2：找入场区 — 候选价位

打开：**Fib + 强压/支 + 通道**

罗列所有候选支撑/压力价位。

#### Step 3：共振过滤 — 高质量价位

打开：**共振 RESO**

只看右上角面板里 **×3 以上**且**距现价 ±5% 内**的价位。

#### Step 4：扣扳机 — 等形态确认

打开：**形态 PAT**

价格回撤到共振区时：
- 出现**锤子线 / 看涨吞没** → **买入扣扳机**
- 反弹到共振压力区出现**射击之星 / 看跌吞没** → **卖出扣扳机**
- 没等到形态信号 → 继续观望

### 6.3 中长线"三维启动"工作流

**三维启动是中长线主升浪逻辑驱动**，不适合 T+0 短线。

| 阶段 | 操作 |
|---|---|
| 候选阶段 | 找候选发现 Stage 1/2 票 → 加自选 |
| 等待阶段 | 自选页每日扫描 → 关注哪些票"距突破 < 1.5%" |
| 突破日 | 找发车扫描显示 S 级 → 追涨买点（突破 K 收盘价 +1%）介入 1/3 仓 |
| 回踩日 | 价格回踩黄金买点（蓄势上沿附近）→ 加仓到 2/3 |
| 加仓日 | 价格突破 MA5 后再放量 → 加仓到满仓 |
| 减仓 / 止损 | 跌破 MA20 减仓；跌破蓄势下沿 ×0.97 全部止损 |

### 6.4 推荐组合（按场景）

| 场景 | 必开 | 可选 |
|---|---|---|
| **个人复盘标配**（普通用户）| 均线 + MACD + KDJ | BOLL |
| **找突破机会** | 箱体 + 强压/支 + 共振 | MACD |
| **跟踪持仓** | 趋势线 + 通道 + 共振 | 形态、缺口 |
| **短线做 T** | **共振 + Fib + 形态**（核心三件套） | 强压/支 |
| **判断顶底** | 波段 + 形态 + 共振 | 强压/支 |
| **中长线主升浪** | **三维启动 + 启动分 + MACD** | 均线、缺口 |

### 6.5 信号可信度排序（弱 → 强）

1. 单一信号
2. 同类共振（两条压力对齐）
3. 多类共振（共振面板 ×2）
4. 多类共振 + 形态（共振面板 ×3 + 锤子/吞没）
5. **三维启动 fresh + S 级 + 大盘强势**（最强组合）

## 7. 局限与注意事项

1. **算法基于历史数据** — 不能预测未来，只能识别已经形成的形态
2. **A 股特殊性** — 涨跌停制度、消息驱动会让纯技术分析失灵
3. **复盘工具，不是交易系统** — 决策应结合基本面、消息面、资金面
4. **算法参数没有"最优值"** — 不同股票不同周期可能需要微调
5. **多指标共振 ≠ 必然方向** — 仍可能反向，但概率上更可靠
6. **形态信号必须看位置** — 锤子线在压力区是无意义的
7. **共振区也会被突破** — 突破后该位置反转角色（支撑 ↔ 压力）
8. **三维启动有趋势位置过滤** — 长期阴跌的票不会进入候选名单（设计意图，避免假蓄势）
9. **找候选/找发车依赖 pytdx** — 北交所（4xxx/8xxx/920xxx）股票不被支持，会自动跳过
10. **S 级评级 ≠ 必赚** — 评级是"形态完美度"指标，不保证未来涨跌；只代表"按规则介入胜率较高"
