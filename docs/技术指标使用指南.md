# K 线技术分析指标使用指南

> 本工具内置 11 项自动技术分析功能：压力/支撑（简单+聚类）、箱体、平台、趋势线、波段、通道、斐波那契回撤、K 线形态、共振检测。
> 本文说明各自原理、用法及组合策略。

---

## 速览

| 指标 | 视觉形式 | 颜色与线型 | 表达内容 |
|---|---|---|---|
| 简单压力/支撑 | 横向直线 | 红虚线 / 绿点线（细） | 近 N 根 K 线的最高/最低 |
| **聚类强压/支** | 横向粗实线 | 红实线 / 绿实线（粗） | **被反复试探**的关键档位 |
| 箱体 | 半透明矩形 | 蓝紫色 + 虚线边 | 持续震荡区间 |
| 平台 | 半透明矩形 | 青绿色 + 实线边 | 缩量横盘的强势整理区 |
| 趋势线 | 斜线 | 蓝色实线 / 橙色虚线 | 上升支撑线 / 下降压力线 |
| 波段 | 折线骨架 + 节点圆点 | 蓝实线段 / 橙虚线段 | 价格主结构 |
| 通道 | 平行四边形 | 青色边 + 半透明填充 | 价格运行的"轨道"上下界 |
| **斐波那契回撤** | 7 条横向线 | 玫红实虚相间 | 主波段的 0/23.6/38.2/50/61.8/78.6/100 % 回撤位 |
| **K 线形态** | 箭头 + 文字标注 | 蓝↑（多）/ 橙↓（空）/ 灰□（中性） | 锤/星/吞/十 5 种敏感形态 |
| **⚡ 共振区** | 右上角浮动面板 | 红边框醒目卡片 | **多源信号汇聚的高质量价位**（最关键） |

色弱友好原则：所有指标统一通过**颜色 + 线型 + 形状**三重冗余编码。
- 蓝 = 上行 / 升势 / 多头形态 / 支撑
- 橙 = 下行 / 跌势 / 空头形态 / 压力
- 实心圆 = 高点，空心圆 = 低点
- ↑ 三角 = 看涨形态，↓ 三角 = 看跌形态

## 一、压力/支撑（两种模式）

### 简单版（SR）— 默认

近 N 根 K 线的最高 = 压力，最低 = 支撑，各一条。

| 周期 | N（窗口） |
|---|---|
| 日 K | 60 根（约 3 个月） |
| 周 K | 26 根（约半年） |
| 月 K | 12 根（一年） |
| 年 K | 5 根 |

带防"插针"过滤：候选高/低必须有 ±2 根内的相邻 K 线在 ±1.5% 价差内"认可"，否则跳到次优。

### 聚类版（SR_K）— 高级模式

跟简单版互补的智能版：滑窗找所有 swing 高/低 → 价格 ±1.5% 内归并 → 至少触及 2 次 → 时间衰减打分 → 离现价 > 25% 剔除 → 上下各取 Top 3。

**视觉对比**：

| | 简单版 | 聚类版 |
|---|---|---|
| 线宽 | 1px | 2px（粗，更醒目） |
| 线型 | 虚线/点线 | 实线 |
| 标题 | `压·近60` | `强压·×7`（带触及次数） |
| 含义 | 近期边界 | **历史多次被验证的真档位** |

**两者可同时打开**：重合的位置 = 超强档位（共振面板会自动识别为多源共振）。

### 实战用法

| 场景 | 解读 |
|---|---|
| 价格上涨触及压力 | 可能反转或调整 |
| 价格下跌触及支撑 | 可能反弹 |
| 突破压力 + 放量 | 上行延续信号 |
| 跌破支撑 + 放量 | 下行加速信号 |
| 简单+聚类压力**同时**显示在某价位 | **超强压力**，反转概率高 |
| 现价已是窗口最高/最低 | 该侧不显示线（突破/创新低中） |

## 二、箱体

滑窗扫描，连续 ≥8 根 K 线满足 `(高-低)/中位 ≤ 12%` → 视为震荡区。贪心向后扩展，直至破位或达到 80 根。

**实战用法**：
- 价格在箱内：高抛低吸，贴近上沿减仓、贴近下沿加仓
- 突破箱顶 + 放量：突破信号，目标 ≈ 箱顶 + 箱高
- 跌破箱底 + 放量：破位信号，下行参考 ≈ 箱底 - 箱高
- 长期窄幅：越窄越久 → 突破后能量越强

---

## 三、平台

「箱体的加强版」，必须同时满足：
1. 时间足够长：≥ 20 根
2. 振幅足够窄：≤ 10%
3. **量能确认**：平台内均量 / 平台前 30 根均量 ≤ 0.85（缩量 ≥ 15%）

代表"价格横盘 + 资金观望或暗中吸筹"，比纯箱体更有意义：
- 平台越长 + 缩量越明显 → 突破后爆发力越强
- **带量突破**才可靠
- 突破后参考"平台高度"作为目标价

## 四、趋势线

1. 滑窗（±5 根）找 swing 高/低
2. 同类型 swing 两两连线生成候选
3. 校验：从起点到当前 K 线无任何 bar 突破/跌破（容差 ±1.2%）
4. 触及 ≥ 2 次 + 未破位 → 按"触及数 × 时间跨度"打分
5. 上升（连低点）/ 下降（连高点）各取最强 1 条

**视觉**：
- **蓝色实线**（连低点 = 上升趋势线 = 支撑）
- **橙色虚线**（连高点 = 下降趋势线 = 压力）

**实战**：
- 蓝线回踩 → 潜在买点；有效跌破 → 上升趋势告破
- 橙线反弹 → 潜在卖点；有效突破 → 下降趋势可能反转
- 突破后压力线常反转成支撑（量能确认时尤其有效）
- 两条都出现 = 三角形整理，收敛终将选择方向

---

## 五、波段（Zigzag）

1. 大窗口（±10 根）找 swing 高/低，过滤小噪声
2. 高低交替化：连续两个同类型 swing 只保留更极端的
3. 输出严格"高→低→高→低"交替的折线节点

**视觉（分段着色）**：
- 上行段（low→high）：蓝色实线
- 下行段（high→low）：橙色虚线
- 节点：高点 = 橙色实心圆，低点 = 白底蓝边

**实战**：
- 看清主结构 / 确认当前在第几波
- 历史每段涨跌幅可作为下一段参考
- 连续创新高但 MACD/KDJ 走低 = 顶背离信号

---

## 六、趋势通道（Raff 平行通道）

1. 取最近 zigzag 的 swing 高 + swing 低
2. 分别 LR 拟合得 slope，取均值作为通道统一斜率
3. 上轨过最高 swing 高（包络）、下轨过最低 swing 低
4. **上下严格平行**

**实战**：
- 贴上轨 = 高位（潜在卖点 / 减仓）
- 贴下轨 = 低位（潜在买点 / 加仓）
- 突破上轨 = 趋势加速向上（带量更可靠）
- 跌破下轨 = 趋势加速向下
- 通道斜率 = 主趋势方向与速度

## 七、斐波那契回撤（短线核心工具之一）

### 原理

1. 复用 zigzag 波段节点
2. 找相邻两个 swing 中**振幅最大**的一段（要求 ≥ 10% 振幅）
3. 自动判方向：起点 < 终点 = 上涨回撤；反之 = 下跌反弹
4. 画 7 条经典回撤线：**0% / 23.6% / 38.2% / 50% / 61.8% / 78.6% / 100%**

### 视觉

- 关键档（0% / 50% / 100%）：玫红 `#ec4899` 粗虚线
- 中间档（23.6% / 38.2% / 61.8% / 78.6%）：淡粉 `#f9a8d4` 点线
- 右轴 label：`Fib 38.2%` 等

### 实战用法

**上涨回撤场景（最常见）**：股价从低点拉升到高点后回调，会在哪一档止跌？

| 回撤档位 | 含义 |
|---|---|
| **23.6%** | 浅回撤，强势行情 |
| **38.2%** | 多头健康调整（黄金买点之一） |
| **50%** | 中性，半数回撤 |
| **61.8%** | 深度回调（黄金分割位，最经典买点） |
| **78.6%** | 极深，多头转空风险 |
| **100%** | 已退回起点，趋势可能终结 |

**下跌反弹场景**：跌幅一段后反弹会被卡在哪个回撤位？同样 5 档作为反弹压力位。

### 短线使用核心

斐波那契**最强用法 = 与其他工具共振**：
- Fib 38.2% **+** MA20 同价位 → 强支撑
- Fib 61.8% **+** 通道下轨 同价位 → 超强支撑
- Fib 50% **+** 上升趋势线 同价位 → 教科书买点

共振检测会自动识别这些位置（见下文）。

## 八、K 线形态识别（短线入场触发器）

### 5 种敏感形态

| 形态 | 算法判定 | 信号 | 视觉 |
|---|---|---|---|
| **锤子线** | 下影 > 实体×2.2，上影 < 实体×0.4 | 底部反转买入 | 蓝色 ↑ 三角（K 线下方）+ 文字「锤」 |
| **射击之星** | 上影 > 实体×2.2，下影 < 实体×0.4 | 顶部反转卖出 | 橙色 ↓ 三角（K 线上方）+ 文字「星」 |
| **看涨吞没** | 前阴后阳 + 后实体完全包裹前实体 | 反转向上 | 蓝色 ↑ 三角 + 文字「吞」 |
| **看跌吞没** | 前阳后阴 + 后实体完全包裹前实体 | 反转向下 | 橙色 ↓ 三角 + 文字「吞」 |
| **十字星** | 实体 < 振幅×8% | 犹豫/反转预警 | 灰色 □ + 文字「十」 |

### 实战用法

形态信号**单独看不够**，必须结合位置：

| 形态出现位置 | 含义 |
|---|---|
| 锤子线 + 在支撑/Fib 位/通道下轨 | **强买入信号** |
| 射击之星 + 在压力/Fib 位/通道上轨 | **强卖出信号** |
| 看涨吞没 + 多次试探支撑后 | **底部反转确认** |
| 看跌吞没 + 多次试探压力后 | **顶部反转确认** |
| 十字星 + 在趋势末端 | **趋势可能反转预警** |

形态出现在**共振区**时（多源信号汇聚的位置） → 信号可靠度倍增。

## 九、⚡ 共振检测（短线最关键功能）

### 原理

把所有活跃指标产生的"当前关键价位"汇集起来，做价格聚类，找出**多个独立来源汇聚到同一档位的"共振区"**。

**算法**：
1. 收集各活跃指标的当前价位：
   - 简单 S/R 压力/支撑值
   - 聚类强压/支位
   - 通道上下轨在最新 bar 的价格
   - MA5 / MA10 / MA20 / MA60 / BOLL 上中下的最新值
   - 箱体顶/底、平台顶/底
   - 上升/下降趋势线在最新 bar 的外推价格
   - 斐波那契 0% / 38.2% / 50% / 61.8% / 100% 价位
2. 价差 ≤ 1.2% 内归并到同一档
3. 不同 source 数 ≥ 2 才算共振
4. 按 source 数倒序排，取 Top 6

### 视觉

主图右上角红边框醒目浮动面板：
- 顶部：`⚡ 共振区 N 处`
- 每条：价格 + 距现价 ±X% + ×N（来源数）+ source 列表
- 红色价格 = 高于现价（潜在压力共振区）
- 绿色价格 = 低于现价（潜在支撑共振区）

### 实战用法（短线核心方法论）

**为什么共振是杀手级功能**：单一指标信号容易出现假信号，多源指标汇聚到同一价位 = 不同分析框架对同一结论的独立验证。短线最怕假信号，共振是最有效的过滤器。

**典型共振场景**：

| 共振组合 | 解读 |
|---|---|
| 简单支撑 + Fib 50% + MA20 | 三重支撑，回踩到此位置 = 高胜率买点 |
| 强压 + Fib 61.8% + 通道上轨 | 三重压力，反弹到此位置 = 高胜率卖点 |
| 通道下轨 + 上升趋势线 + 平台底 | 三重支撑，主趋势未破 |
| 压力 + 下降趋势线 + 通道上轨 | 三重压力，反转概率高 |

**操作要点**：
- ×3 以上的共振区 = 强信号
- ×2 已经有意义，但需要其他确认（量能 / 形态）
- 距现价 ±5% 内的共振区最有交易价值
- 距现价 > 10% 的共振区作为远期参考

## 组合用法（最重要的部分）

### 三种典型行情匹配指标

#### 1. 趋势行情（单边上升或下跌）

主力工具：**趋势线 + 通道 + Fib**
辅助工具：**波段、共振**
不必使用：箱体 / 平台

#### 2. 横盘整理

主力工具：**箱体 + 平台 + 简单 S/R**
辅助工具：**强压/支、共振**
不太适用：趋势线 / Fib / 通道

#### 3. 转折期（不确定）

主力工具：**波段、形态、共振**
辅助工具：**趋势线、Fib、强压/支**

---

### 短线（看长做短）四步法 ⭐⭐⭐

短线交易的标准工作流，把工具按目的分层：

#### 第一步：看长 — 确认主方向

打开：**通道 + 趋势线 + 波段**

判断：
- 通道斜率向上 + 蓝色趋势线还在 = 上升趋势 → **只做多不做空**
- 通道斜率向下 + 橙色趋势线还在 = 下降趋势 → **只做空或观望**
- 横盘 / 趋势模糊 → 跳过这只票

#### 第二步：找入场区 — 候选价位

打开：**Fib + 强压/支 + 通道**

罗列所有候选支撑/压力价位。

#### 第三步：共振过滤 — 高质量价位

打开：**共振**

只看右上角面板里 **×3 以上**且**距现价 ±5% 内**的价位。

#### 第四步：扣扳机 — 等形态确认

打开：**形态**

价格回撤到共振区时：
- 如果该共振区出现**锤子线 / 看涨吞没** → **买入扣扳机**
- 如果反弹到共振压力区出现**射击之星 / 看跌吞没** → **卖出扣扳机**
- 没等到形态信号 → 继续观望，不强行入场

### 推荐组合（按场景）

| 场景 | 必开 | 可选 |
|---|---|---|
| **个人复盘标配** | 波段 + 简单 S/R | 形态 |
| **找突破机会** | 平台 + 通道 + 简单 S/R | 量能（看 K 线 vol 颜色） |
| **跟踪持仓** | 趋势线 + 通道 + 共振 | 形态 |
| **短线做 T** | **共振 + Fib + 形态**（核心三件套）| 强压/支 |
| **判断顶底** | 波段 + 形态 + 共振 | 强压/支 |

---

### 信号可信度排序（弱 → 强）

1. 单一信号
2. 同类共振（两条压力对齐）
3. 多类共振（共振面板 ×2）
4. 多类共振 + 形态信号（共振面板 ×3 + 锤子/吞没）
5. 多类共振 + 形态 + 量能确认（缩量平台后突破 + 强压共振 + 看跌吞没）

## 参数调整指南

### 不同周期的合理设置

| 参数 | 日 K | 周 K | 月 K |
|---|---|---|---|
| swing pivotWindow | 5 ~ 10 | 3 ~ 5 | 2 ~ 3 |
| 通道 maxPivots | 5 | 4 | 3 |
| 箱体 minBars | 12 ~ 20 | 8 ~ 12 | 6 ~ 8 |
| 平台 minBars | 20 ~ 30 | 12 ~ 20 | 8 ~ 12 |
| Fib minRangePct | 10% | 15% | 20% |
| 形态 recencyBars | 200 | 100 | 50 |

### 不同股票特性

| 股票类型 | 调参方向 |
|---|---|
| 大盘股 / 蓝筹 | 波动小，箱体阈值收紧（8% 而非 12%） |
| 中小盘 / 题材 | 波动大，阈值放宽（15-20%） |
| 次新股 | "近 N 根"参数缩短 |
| 长期慢牛 | pivotWindow 调大（看大波段） |
| 暴涨暴跌 | pivotWindow 调小，及时识别新结构 |

## 局限与注意事项

1. **算法基于历史数据** — 不能预测未来，只能识别已经形成的形态
2. **A 股特殊性** — 涨跌停制度、消息驱动会让纯技术分析失灵
3. **复盘工具，不是交易系统** — 决策应结合基本面、消息面、资金面
4. **算法参数没有"最优值"** — 不同股票不同周期可能需要微调
5. **多指标共振 ≠ 必然方向** — 仍可能反向，但概率上更可靠
6. **形态信号必须看位置** — 锤子线在压力区是无意义的，必须在支撑区或共振区才有交易价值
7. **共振区也会被突破** — 突破后该位置反转角色（支撑变压力 / 压力变支撑）
